Pour pouvoir utiliser notre programme il faut : 
- Importer le projet dans eclipse (File -> import...).
- Ajouter les projets chat.messages dans le Build Path du projet.

Une fois le projet récupéré, ajoutez comme Application Java le fichier TestConnectionBurst.java. Ce programme
connecte 5 machines les unes avec les autres. Une fois connectées, elles s'envoient des messages entre elles.
Vous pouvez ajouter en argument d'application le port de base à utiliser pour les connexions. Si vous mettez le port
20000 les machines auront les ports suivants : 20000, 20001, 20002, 20003, 20004.
